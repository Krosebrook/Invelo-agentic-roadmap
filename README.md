# Agentic Intelligence Portal

A strategic intelligence dashboard synthesizing the 2026 enterprise paradigm shift towards autonomous agentic infrastructure, modular skill architectures, and orphaned demand market opportunities.

## 🚀 Overview

The **Agentic Intelligence Portal** is designed for staff engineers, architects, and strategists to navigate the complex landscape of 2026 AI deployment. It "weaves" together high-level strategic memos with actionable market data and technical configuration tools.

### Key Modules

- **Synthesis Weave**: A visual graph connecting infrastructure, execution, and market threads.
- **Strategic Memos**: Full-text analysis of the "Transplantable Skeleton," "SaaS Apocalypse," and "Constitutional AI."
- **Market Opportunities**: A target-rich list of "Orphaned Demand" concepts with monetization and data extraction models.
- **Knowledge Base**: A high-fidelity ingestion pipeline for documents, spreadsheets, and archives.
- **Agent Configuration**: Fine-tuning cognitive parameters (temperature, tokens) and security boundaries.
- **Historical Weaving**: A technical and historical exploration of textile structures (Tabby, Twill, Satin) applied to digital pattern generation.

## 🛠 Tech Stack

- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Parsing**: JSZip (Archives), PapaParse (CSV)
- **Typography**: Playfair Display (Serif), Inter (Sans), JetBrains Mono (Mono)

## 📁 Project Structure

- `src/components/`: Modular UI views for each dashboard section.
- `src/data.ts`: Centralized repository for all strategic content and market data.
- `src/lib/utils.ts`: Core utility functions (e.g., `cn` for Tailwind merging).
- `src/index.css`: Global theme definitions and font imports.

## 🧠 Design Philosophy

The portal adheres to a "High-Fidelity Executive" aesthetic, prioritizing legibility, structural coherence, and subtle micro-interactions. It uses a dark-mode palette (`zinc-950`) to minimize eye strain during deep strategic analysis.
